O que é semântica?

Semântica em HTML significa usar tags que descrevem o conteúdo de forma clara. Tags como <header> e <footer> ajudam a entender a função do conteúdo.

Por que é importante?

  • Acessibilidade: Facilita a leitura para tecnologias assistivas.
  • Clareza: Torna o código mais organizado e fácil de manter.
